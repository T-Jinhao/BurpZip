#### 压缩包爆破

##### 依赖库

- zipfile



##### 使用

`python BurpZip.py -f [file] -e [*payloadfile]`

有密码则出现操作提示



##### 实现流程

读取文件，判读是否为zip压缩文件

尝试使用空密码（模拟无密码）解压，失败则进入有密解压。

用户可输入密码，也可进行爆破。

判断是否传入密码本，若有则直接使用payload

若无则自行选择字典及密码长度，脚本开始爆破。

爆破成功，输出密码。

##### 改动
